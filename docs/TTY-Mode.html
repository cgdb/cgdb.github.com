3.4 Commands available during TTY mode

cgdbmodekey

Switch back to source window. This is defaulted to the ESC key.

page up

Move up a page.

page down

Move down a page.

F11

Go to the beginning of the GDB buffer.

F12

Go to the end of the GDB buffer.
